ai-vid-edit

AI-powered podcast and interview video editor. Automate scene detection, transcription, and intelligent video processing with Python.

Download Latest

Features

  • AI scene detection — Automatic identification of scene boundaries
  • Transcription — AI-powered speech-to-text for podcasts
  • Web UI — Flask-based interface for easy upload and processing
  • CLI — Scriptable processing pipeline
  • Batch processing — Handle multiple videos efficiently

Quick Start

cd podcast-video-editor
uv sync
uv run python -m podcast_video_editor --help
